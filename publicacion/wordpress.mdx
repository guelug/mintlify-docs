La culminación del flujo de trabajo es la publicación. Hemos integrado una herramienta que conecta directamente con los portales de noticias de la red FUNIBER.

## Pasos para Publicar

### 1. Preparación final
Antes de publicar, asegúrate de que el título y el cuerpo del texto en el editor sean los definitivos. No olvides revisar el **SEO** para que la noticia vaya optimizada desde el primer segundo.

### 2. Abrir el Panel de Publicación
Haz clic en el botón **"Publicar en WP"** (normalmente identificado con el logo de WordPress W).

### 3. Configuración del Post
Se abrirá un cuadro donde podrás:
- **Verificar el Título**: Es el que saldrá en la web.
- **Categorías**: Dependiendo del portal, verás una lista de categorías (ej: *Institucional*, *Becas*, *Convenios*). Selecciona la adecuada.
- **Etiquetas (Tags)**: La IA te sugerirá algunas; puedes borrarlas o añadir las tuyas.

### 4. Envío como Borrador
Haz clic en **"Enviar a Borrador"**. 
**¡Importante!**: La noticia NO se publica directamente al público. Se guarda en el WordPress oficial como un "Borrador" para que un editor humano le dé el visto bueno final, asigne la imagen destacada y programe la hora de publicación.

## Ventajas del sistema:
- **Adiós a los errores de formato**: El sistema limpia el código HTML para que se vea perfecto en la web.
- **Sincronización Total**: Título, texto y metadatos se envían en un solo bloque.
